# resume-site

這是給 GitHub Pages 使用的個人履歷網站檔案。

## 檔案說明

- `index.html`：網站主頁內容
- `style.css`：網站樣式
- `resume.pdf`：完整履歷 PDF
- `robots.txt`：降低搜尋引擎索引機率
- `.nojekyll`：讓 GitHub Pages 直接發布靜態檔案

## 上傳到 GitHub 的方式

1. 進入你的 `resume-site` repository。
2. 點選 `Add file` → `Upload files`。
3. 將本資料夾內的所有檔案上傳到 repository 根目錄。
4. 點選 `Commit changes`。
5. 進入 `Settings` → `Pages`。
6. Source 選 `Deploy from a branch`。
7. Branch 選 `main`，Folder 選 `/root`。
8. 點選 `Save`，等待 GitHub Pages 產生網站連結。

## 注意

此網站已加入 noindex 與 robots.txt，但這不等於密碼保護。網站連結仍可能被轉傳，建議只提供給特定對象。
